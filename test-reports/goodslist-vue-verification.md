## 🔧 修正した問題

### 問題1: UIテキスト言語切り替えテストの失敗

**症状:**
- `page.goto('/?lang=ja')`でページ遷移後、Vueコンポーネントの初期化が完了する前にテストが実行される
- 結果：日本語モードなのに「EN」が表示される

**原因:**
- Vueコンポーネントの`onMounted`がI18nManagerの初期化完了前に実行される
- 非同期初期化のタイミング問題

**修正内容:**
```typescript
// tests/e2e/language-toggle.spec.ts (121行目前)
await page.waitForTimeout(300); // Vueコンポーネントの初期化を待つ
```

**結果:** ✅ テスト成功

## 🏗️ アーキテクチャ

### Vueコンポーネントの統合方式

```
entry.ts
  ├─ App.initialize() (I18nManager初期化)
  ├─ initVueComponents() (LanguageToggle.vue)
  └─ initGoodsList() (GoodsList.vue) ※USE_VUE_GOODS_LIST=true時
```

### 機能フラグによる段階的移行

```typescript
// entry.ts
const USE_VUE_GOODS_LIST = true;  // trueでVue版、falseで旧版
```

この方式により：
- ✅ 新旧コンポーネントの共存が可能
- ✅ 問題発生時の即座のロールバック
- ✅ 段階的な移行とテスト
