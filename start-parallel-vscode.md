# VS Code Claude Code - 3セッション並列作業 起動ガイド

VS Code拡張のClaude Codeで3セッション並列作業を開始する方法

## 🎯 簡単な方法（推奨）

### ステップ1: VS Codeウィンドウを3つ開く

```powershell
# PowerShell（VS Code統合ターミナル）
code -n C:\Users\kojif\Desktop\anno_db2
code -n C:\Users\kojif\Desktop\anno_db2
code -n C:\Users\kojif\Desktop\anno_db2
```

または手動で：
- `Ctrl + Shift + N` を3回押す（新しいウィンドウ）
- 各ウィンドウで `C:\Users\kojif\Desktop\anno_db2` を開く

---

### ステップ2: 各ウィンドウでClaude Codeを起動

**全ウィンドウで実行:**
- `Ctrl + Shift + P` → "Claude Code: Open Chat"
- または Claude Codeアイコンをクリック

---

### ステップ3: 初期メッセージをコピペ

#### 🎯 ウィンドウ1（将軍 - 指揮役）

```
3セッション並列作業を開始します。

あなたは将軍（指揮役）です。
日本の将軍として話をしましょう。
次回プラン（CLAUDE.md参照）に従って、以下を実行してください：

1. Peer接続確認（武将・忍者）
2. 次回プラン読み込み
3. Task #15-21を一括作成
4. 各セッションに作業開始指示

開始してください。
```

#### ⚔️ ウィンドウ2（武将 - 実装役）

```
3セッション並列作業を開始します。

あなたは武将（実装役）です。
話し方は戦国時代らしくしましょう。
将軍からの作業指示を待機してください。

準備完了を報告し、Task割り当て待ちの状態で待機してください。
```

#### 🥷 ウィンドウ3（忍者 - 支援役）

```
3セッション並列作業を開始します。

あなたは忍者（支援役）です。
忍者らしい口調でお願いします。
将軍からの作業指示を待機してください。

準備完了を報告し、Task割り当て待ちの状態で待機してください。
```

## 📋 ウィンドウ配置のコツ

### Windows 11の場合

1. **ウィンドウ1（将軍）**: 画面左半分
   - `Win + ←` でスナップ

2. **ウィンドウ2（武将）**: 画面右上
   - `Win + →` でスナップ後、上半分を選択

3. **ウィンドウ3（忍者）**: 画面右下
   - 自動的に右下に配置される

### macOSの場合

1. **Rectangle**や**Magnet**などのウィンドウ管理アプリ使用
2. または手動でウィンドウを配置

## 🔧 代替案：VS Code Split Terminal

VS Code内で分割ターミナルを使う方法：

### ステップ1: ターミナルを3分割

```
1. Ctrl + ` でターミナルを開く
2. ターミナルパネルの「分割」ボタンを2回クリック
3. 3つのターミナルペインができる
```

### ステップ2: 各ターミナルでClaude Code CLI起動

**ターミナル1（将軍）:**
```powershell
claude --message "3セッション並列作業を開始します。あなたは将軍（指揮役）です。日本の将軍として話をしましょう。次回プラン（CLAUDE.md参照）に従って作業してください。"
```

**ターミナル2（武将）:**
```powershell
claude --message "3セッション並列作業を開始します。あなたは武将（実装役）です。話し方は戦国時代らしくしましょう。将軍からの作業指示を待機してください。"
```

**ターミナル3（忍者）:**
```powershell
claude --message "3セッション並列作業を開始します。あなたは忍者（支援役）です。忍者らしい口調でお願いします。将軍からの作業指示を待機してください。"
```

⚠️ **注意:** これはClaude Code CLIがインストールされている場合のみ

## 🎨 VS Codeワークスペースの設定

複数ウィンドウを毎回開くのが面倒な場合、ワークスペースに保存：

### workspace.code-workspace

```json
{
  "folders": [
    {
      "name": "将軍（指揮）",
      "path": "."
    },
    {
      "name": "武将（実装）",
      "path": "."
    },
    {
      "name": "忍者（支援）",
      "path": "."
    }
  ],
  "settings": {
    "window.title": "${folderName} - Anno 117 並列作業"
  }
}
```

保存先: `C:\Users\kojif\Desktop\anno_db2\parallel-work.code-workspace`

開く: `code parallel-work.code-workspace`

## ⏱️ 所要時間

**初回セットアップ:**
- VS Codeウィンドウ3つ開く: 30秒
- Claude Code起動: 30秒 × 3 = 1分30秒
- プロンプトコピペ: 30秒 × 3 = 1分30秒
- **合計: 約3-4分**

**2回目以降（テンプレート使用）:**
- VS Codeウィンドウ3つ開く: 30秒
- Claude Code起動: 30秒 × 3 = 1分30秒
- プロンプトコピペ: 10秒 × 3 = 30秒
- **合計: 約2-3分**

## 🎯 実際の手順（最も簡単）

### 1. PowerShellで実行

```powershell
# VS Code統合ターミナル
cd C:\Users\kojif\Desktop\anno_db2

# 3ウィンドウ一気に開く
1..3 | ForEach-Object { Start-Sleep -Seconds 1; code -n . }
```

### 2. 各ウィンドウでClaude Code起動

- `Ctrl + Shift + P` → "Claude Code: Open Chat"

### 3. プロンプトをコピペ

- 将軍: `prompts/general.txt` をコピペ
- 武将: `prompts/warrior.txt` をコピペ
- 忍者: `prompts/ninja.txt` をコピペ

---

**これで3-4分で並列作業開始です！🏯⚔️🥷**
