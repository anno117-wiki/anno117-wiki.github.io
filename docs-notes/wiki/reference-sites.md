## 活用メモ

- **生産チェーンデータ**: `anno-calculator.org` と `anno.land` が最も詳細。比率・建物数ともに信頼性高い
- **人口Tier情報**: Fandom Wiki と GameRant が網羅的。Latium 4Tier / Albion 別Tierの差異に注意
- **日本語コンテンツ**: 独立したwikiは薄い。ゲームウィキ.jpとナムウィキが現状最も実用的
- **レイアウトTips**: GamesRadar と GAMES.GG が図解付きで参考にしやすい
- **公式情報**: Ubisoft公式Tipsは初心者向けに5項目を簡潔にまとめており、Wiki冒頭に引用しやすい
