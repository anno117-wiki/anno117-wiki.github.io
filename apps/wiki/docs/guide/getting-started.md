# はじめに

Anno 117 統合Wiki へようこそ。本サイトは Anno 117（PS5 / Steam）の日本語情報Wiki と、生産チェーン計算機を統合した Web アプリケーションです。

## 構成

本サイトは二つの部分から成ります。

- **Wiki（本サイト）** — 商品・生産・地域などの情報を日本語で整理した解説ページ群
- **計算機** — 生産チェーンを展開し、必要な建物数・原料を算出する専用ツール

## 計算機を使う

生産チェーンの計算を行うには、ナビゲーションの「計算機」、またはトップページの「計算機を開く」から計算機ページへ移動してください。

計算機では、商品をツリーから選び、生産チェーンと必要量を確認できます。

## データの出典

商品・生産チェーンのデータは、anno-calculator公式（GitHub: agentquackyt/Anno117Calculator）のデータを活用しています。
