交易・交易ルートガイド ​

このページはインターネット上の攻略情報を集約した内容です。未検証の情報を含む場合があります。

交易の全体像 ​

Anno 117 の交易には大きく2種類あります。

  • 能動的交易（交易ルート）: 自分の船を使い、島と島（および地域間）を巡回させて商品を自動で積み下ろしします。自分の島同士でも、NPC（他都市）相手でも設定可能です。
  • 受動的交易: 交易所の各商品に「売り設定・買い設定」を設定しておくと、交易協定を結んだ相手やNPC商人が自動で船を寄越して取引します。手動管理がほぼ不要です。

交易ルートの設定手順（能動的交易） ​

  1. 画面下部のHUDから交易ルートメニューを開く。アイコンは「2本の矢印」。
  2. 「新規交易ルート」を選択。
  3. **1つ目の島（出発地）**を選ぶ。
  4. **2つ目の島（目的地）**を選ぶ。3島以上を順に追加することも可能。
  5. ルートに使う船を割り当てる。
  6. 各島で積む商品／下ろす商品を設定する。商品ごとにカーゴスロットを割り当てる。
  7. 「確定」でルートを起動。以後、船がルートをループ巡回し自動で積み下ろしします。

地域をまたぐルート（ラティウム ⇄ アルビオン） ​

  • 基本手順は同じですが、島を選ぶ際に地域マップを切り替える必要があります。
  • 出発地・目的地は異なる地域でも指定できます。

受動的交易（売買注文） ​

交易所・港のメニューで各商品に閾値を設定します。

  • 最低在庫: 常に島に残しておく量。これ以下は売らない。
  • 上限値: この量を超えると自動で売却を開始する。
  • 下限値: この値を下回ると商品を、上限まで自動で買い付ける。
  • 交易協定を結んだ相手は、定期的に船を寄港して売買してくれます。

価格・収益の注意 ​

  • 各商品には金銭価値に基づく単価があり、メニュー上のアイコンに表示されます。
  • 受動的交易の単価は控えめです。
  • NPCは余剰生産を勝手に買い取らない。売り設定を能動的に行う必要があります。
  • NPCは自分が不足している商品のみ購入します。
  • ゲーム設計上、過剰生産での輸出は労働者コストに見合わないことが多く、輸出特化より自分の都市の需要に合わせた生産が推奨されます。
  • 生産過剰により住民の消費が止まるより、売却して経済を回すほうが有益な場合もあります。

船の選択とカーゴ・速度 ​

  • 民間船: カーゴスロットが多く、輸送量に優れます。
  • 軍用船: 攻撃を受けても自衛できます。海賊・敵対勢力対策。
  • 船は3ティア構成: ペンテコンター → トライリーメ → クィンクェレメ。民間・軍団それぞれ存在します。
  • 長距離・大量輸送には五段櫂船を推奨。最良の積載量と耐久力を持ちます。
    • カーゴスロットは最大7スロット（積み下ろし専用3スロット＋モジュール用4スロット）。
    • マストを装備すると風向きによるが最大速度が向上。
    • オールを装備すると風向きによる影響が減少。
    • 船体装甲は船の耐久力を高める。
  • 船の建造は造船所が必要で、木板・ロープ・帆・防具・武器などが必要です。
  • 高ティア船ほどモジュールを多く装着でき、速度増加や弓兵・バリスタによる防御などを付与できます。
  • **専門家（船長）**を船に乗せると交戦力・交易コスト削減など様々な効果を得られます。

NPC・他都市との売買（交易協定・外交） ​

交易協定の締結 ​

  • NPC（ライバル）と取引するにはまず交易協定を結ぶ必要があります。
  • 協定は相手との関係を改善してから利用可能になります。改善手段:
    • クエストの達成
    • 金銭の贈与
    • 島の契約の履行
    • NPC固有のステータスの達成率
  • 協定締結後、新規交易ルートを設定でき、受動的な売買注文も相手の市場に対して有効化されます。

取引相手のキャラクターと商品（例） ​

各キャラは「売りたい商品」と「欲しい商品」を持ちます。例:

  • ドリアン は 大理石／アンフォラ／靴 を売り、チーズ／石鹸／オリーブオイル を求める。
  • ドリアンに オリーブオイル／石鹸／ガルム を売ると、進行に従い最大100個・60〜74/個で買い取り、良い収入源になります。
  • 緊急時には船に余剰を積み、NPC（商人）に直接売って即時資金化できます。
  • NPC（商人）には無制限に購入してくれる商品があります。

外交システム全体 ​

  • NPCライバルを最大3名選択可能。例: ドリアン（健康・文化重視、ラティウムのみ入植、平和志向）、タラゴン（軍人気質）、コンクロディア（宗教・ローマ伝統重視、拡張志向）、アーサー（アルビオンのケルト系統治者）。
  • 外交メニューでは各ライバルの治世状況・条約状況・軍事力/経済力/契約履行・キャラ特性などを確認できます。
  • 取れる外交アクション:
    • 交易協定 — 港での商品売買
    • 防衛協定 — 相互防衛
    • 同盟 — 友好・支援
    • 贈り物（大/小） / 専門家任命（研究が必要）
  • 従属化（属国化）: 同盟状態で相手より強いこと、執政官（執政官代理）の役職を持ち相手を上回る資金（ただし超高額）があると、ライバルを属国化できます。従属したライバルは強力な専門家として自分の総督のヴィラに配置でき、その島は属国都市になります。
  • 戦争: 全面戦争（皇帝の協力を買う）と限定戦争（民間建物と交易ルートを保護）があります。戦争宣言は皇帝への評判を低下させ、評判が低いほど皇帝が介入しやすくなります。
  • 皇帝との関係: 皇帝の要望・行動選択などにより増減し、入植コスト・島の維持費の増減・NPC商人との交易の有無・執政官（執政官代理）の称号取得に影響します。

交易所・港のアップグレード（研究） ​

交易効率を高める主な研究技術:

  • 貯蔵所: 島の保管容量を増やす。
  • 交易所改良: クレーンを追加し積み下ろしを高速化。島の保管容量を増やす。
  • 交易埠頭: 沿岸に配置し積み下ろしができる。航路の距離を短縮できて便利。
  • 港湾指定: 終盤技術。どの船がどの交易所/交易埠頭を使うか指定できる。
  • 修理クレーン: 港で交易船・軍用船を修理。

交易に有用な信仰神 ​

  • ネプトゥーヌス: 序盤向け。新造船の維持費削減・耐久増加・海産物に関する生産チェーンを強化。
  • メルクリウス＝ルグス: 終盤向け。受動的交易の収益と、港/交易所からの名声を強化。受動交易をより儲かるようにします。

交易ルート運用のコツ ​

  • ルート明確化: 島の名前を変える（例: 「樹脂島」「オリーブ島」）。ルート把握が容易になります。
  • 輸出量の調整（往復時間を考慮）:
    • 近い島: 消費量より 2〜3トン多く生産・輸出。
    • 遠い島: 消費が 10トン/分 なら、往復時間を見込み 約15〜16トン/分の積み込み。
    • 最低在庫は低め（50〜100単位）に設定し余剰を輸出に回す。
  • ルートの防衛:
    • 護衛 — 交易船と同速度の軍船を割り当て。
    • 巡回 — 島間に軍船を配置し即応。
    • 迂回 — 海賊多発海域を避けてルート変更。
  • よくある航路の問題:
    • 容量がいっぱいで荷下ろしできない。
    • 在庫がゼロで積み込みできない。
    • 内容だけ変更して以前の荷物が下ろせていない。
    • 船が攻撃を受けて損傷・沈没し機能していない。
