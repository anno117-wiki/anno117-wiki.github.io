研究・スキルツリー・専門家ガイド ​

このページはインターネット上の攻略情報を集約した内容です。未検証の情報を含む場合があります。

研究システムの仕組み（スキルツリー） ​

  • 研究は「スキルツリー」をアンロックしていき、生産効率・選択肢・都市サービスを拡張する仕組み。アンロックした内容は全島で共有されます。
  • ツリーは大きく 経済 / 社会 / 軍事 の3トラック＋DLC「灰の予言」ブランチの計4系統。150以上のスキルが存在します。
  • 排他的な分岐は無い。時間をかければすべてアンロック可能です。
  • 複数の技術を順番にキューへ積める。研究速度は都市が産出する知識量に比例します。

インスピレーション（研究の解放ゲート） ​

  • ツリーの深部を開くには「インスピレーション」の条件を満たす必要があります。条件例: 交易ルート数・水道橋の建設数・都市ステータス。
  • 例として「国際交易ルート5本」「研究ポイント17,000産出」などが研究解放条件として挙がります。

知識ポイントの入手 ​

知識はAnno 117の8つのコア属性のひとつ（人口・収入・信仰・知識・名声・健康・幸福・火災安全）。知識が高いほど研究速度が上がります。

知識を産む主な建物 ​

  • ラテン語学校 — プレブス向けの公共サービス。範囲内の建物に知識 +3。最初に知識を上げる建物のひとつ。住宅密集地の近くに配置し、効果範囲が多くの家をカバーするよう配置するのが推奨です。
  • 図書館 — 大きな知識ボーナスを与えます（知識 +7）。
  • 聖所 — ケルト系の公共サービス建物。信仰 +2 / 知識 +1。序盤に知識を底上げするもうひとつの選択肢。

需要を満たすと増加する知識 ​

  • 筆記板 — エクィテスの需要。供給すると知識 +3。
  • 竪琴 — パトリキの需要。供給すると知識 +2。
  • 手鏡 — パトリキの需要。供給すると知識 +2。

知識を増やすコツ ​

  • 守護神ミネルヴァを島の守護神に設定し「神の英知」を得ると、信仰の数値によりその島の全住民の基礎知識が増加します（最大 +13）。
  • 専門家頭韻の文学者 ホラティア・ヘミナ — 筆記板工房の知識産出 +3。

研究ツリーの種類（トラック別の内容） ​

経済 ​

物流・生産アップグレード・資源アクセス・道路改善。具体内容: 交易インフラ・林業管理・農業灌漑・鉱業強化・倉庫容量拡張など。

序盤推奨ノード:

  • 貯蔵所 — 島の保管容量を増加。基礎ノード。
  • 交易所改良 — 交易所の効率と保管容量を強化・収入増。
  • 裁縫サークル — 糸紡ぎ師の知識産出を上げるため序盤に有用。
  • 舗装道路 — 輸送速度と建物のストリート範囲を増加（手動アップグレードが必要）。
  • 大地の血 — 鉱床の無い島で鉄を低レートで採掘可能にします。
  • 緑地の再生 — 草原に木こりを配置でき、配置の自由度が上がります。
  • 森林再生 — 炭焼き小屋が草原に配置でき、配置の自由度が上がります。

社会 ​

公共サービス・都市範囲/ユーティリティ・修理・維持オプション・生活の質向上。災害対策・市場効果・船の改良・NPC同盟（従属）条約など。

序盤推奨ノード:

  • 市場の力 — 市場の効果範囲を増加。
  • 需品管理 — 船の積荷スロット増・積荷重量減。交易を円滑化し、衛生・防火改善へ繋がります。

軍事 ​

ユニット性能・士気・耐久・遠征支援。兵営・防御構造物・新ユニット（投石兵 / ローマ軍団兵）・海軍アップグレードなど。

  • 軍事は後回し推奨。軍隊は人口・船・資金を要するため、都市が黒字で安定してから着手しましょう。

序盤の全体方針 ​

経済 + 社会を先に進め、軍事は都市が安定してから。現在のボトルネックを解消する1トラックに集中するとよいでしょう。

専門家 ​

前作Anno 1800の「アイテム」が本作では専門家に名称変更。建物や船のスロットに装着して範囲バフを与えるカード的存在です。詳しいデータはアイテム一覧を参照してください。

レアリティ ​

5段階: コモン / レア / エピック / レジェンダリー / ユニーク。高レアほど効果が強く、入手難度も高くなります。

カテゴリ（専門分野） ​

財政 / 宗教 / 研究 / 文化 / 経済 / 自然 / 社会 / 軍事 / 航海

装着先とスロット数 ​

装着先スロット数備考
総督のヴィラ標準5スロット、名声で増加効果範囲が広い。汎用・共通属性バフ向き。住宅地近くに配置。
オフィキウム2スロット → 研究進行で3スロット効果範囲が狭い。建設可能数は島の名声とスキルに依存。ヴィラや他のオフィキウムと範囲を重複させられない。生産系の特化バフ（労働力削減・維持費削減）向きで生産建物の周りに配置。
船専用の船長スロット1つ船長を装着。その船にのみ効果。

効果の仕組み ​

  • 専門家は範囲内の建物数に応じてバフ量が決まります。密集配置ほど効率が上がります。
  • 効果対象: 住宅・生産建物・公共サービス建物・船。
  • 効果の種類: 生産性 +10%〜+35%、労働力 -25%〜-80%、維持費 -25%〜-80%、属性ボーナス（人口・信仰・知識・健康・幸福・火災安全・収入・名声）、特殊効果（例: 10サイクルごとに追加1t）など。

入手方法 ​

  • クエスト/イベント — 専用クエストやイベントの報酬。最初の専門家を無料で1体得られることも。
  • 交易商 — ラティウム / アルビオン等の商人からデナリウスで購入（アイテムタブ）。来訪中の専門家を雇用。
  • スキルツリー — 特定の経済/社会/軍事の研究で専門家をアンロック。
  • 殿堂 — 統計メニュー内。名声ポイントで購入、または実績達成で高レア入手。
  • 祭礼 — 開催するとランダムな専門家・船長がゲストとして確率で来訪。
  • 灯台の契約 — 他プレイヤーの灯台契約を達成すると専門家報酬の抽選チャンス。
